# Components

## App

The main react component of the app. It displays the main user page component: ***UserPage***.

It is composed of the redux provider, which is a predictable state container for JavaScript apps.

## UserFiltersForm

It displays a filters form to filter the users list.

## UserPage

The main user page of the app. It displays the user filters ***UserFiltersForm***, the users list ***UserList*** and the link to the user creation page ***UserForm***.

## UserFiltersForm

The user filters component.

## UserList

It displays all the users he is given in a list.

## UserForm

The user creation page.
